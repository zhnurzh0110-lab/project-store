## Features Implemented
- Semantic HTML structure with header/main/footer on each page
- Responsive layout (media queries at 900px and 600px)
- Form with client-side validation (required fields, email regex, password strength, confirmation)
- Real-time search (filter products)
- Bootstrap modal used for adding items/login
- Dynamic table with Create, Read, Update, Delete (data persisted to `localStorage`)
- Image gallery with category filters and lightbox (images load lazy)
- CSS variables, Flexbox, Grid, :nth-child usage, positioning
- Animations (CSS transitions, jQuery fade/slide)
- (Optional) API integration: FakeStore API
